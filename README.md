# 🇮🇳 India Tourism Data Analysis Dashboard (2014–2020)

## 📊 Project Overview

This project focuses on analyzing India's tourism data from **2014 to 2020** using **Python, SQL, and Power BI**.
The goal is to extract meaningful insights about visitor trends, growth patterns, country-wise contributions, and travel modes.

---

## 🚀 Key Objectives

* Perform **Exploratory Data Analysis (EDA)** on tourism data
* Analyze **visitor growth trends over the years**
* Identify **top contributing countries**
* Study **travel modes (Air, Rail, Sea, Land)**
* Build **interactive Power BI dashboards** for business insights

---

## 🛠️ Tech Stack

* **Python** → Pandas, NumPy, Matplotlib, Seaborn
* **SQL (PostgreSQL)** → Data storage & querying
* **Power BI** → Dashboard & Data Visualization
* **Excel / CSV** → Dataset

---

### 3️⃣ Growth & Performance Dashboard

* YoY Growth Analysis
* Country-wise growth rate
* Contribution % by countries

---

## 🔍 Key Insights

* 📉 Tourism dropped drastically in **2020 due to COVID-19**
* ✈️ **Air travel dominates** tourism in India
* 🌍 Neighboring countries contribute major visitors
* 📊 Growth peaked around **2017–2018**
